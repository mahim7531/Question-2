# Responsive Navbar Website

A simple responsive website template with a mobile-friendly navigation bar.

## Files

- `index.html` - main page markup
- `styles.css` - responsive styling for the navbar and page sections
- `script.js` - mobile menu toggle behavior

## Features

- responsive header navigation
- mobile hamburger menu
- clean layout with multiple sections
- easy to customize and extend

## How to use

1. Open `index.html` in your browser.
2. Resize the browser window to see the navbar adapt.
3. Click the hamburger icon on small screens to open or close the menu.

## Customize

- edit `styles.css` to change colors, spacing, and layout
- edit `index.html` to update navigation links and page content
- edit `script.js` to add additional interactive behavior

## Notes

This is a lightweight starter template ideal for learning responsive design and building a simple landing page.
